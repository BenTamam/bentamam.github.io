## About Me

Doing offensive security work at Check Point Software Technologies, focused on adversary simulation across Windows, Active Directory, and Azure environments. My focus is on designing and refining attack paths that replicate threat-actor TTPs aligned with the MITRE ATT&CK framework.

My work spans red team adversary simulation, vulnerability research, and Windows exploit development. That includes client-facing penetration testing across web, infrastructure, cloud, and thick-client domains, as well as independent security research and CVE discovery.
